# bun + view + typescript で WGPU を練習

## project 初期化

https://ja.vite.dev/guide/

```sh
$ mkdir hello_wgsl
$ cd hello_wgsl
$ bun create vite .
# react + typescript
$ bun install
$ bun run dev
```
